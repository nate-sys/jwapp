# Jwapp
A Journal webapp



## Usage

Build from source
```sh
git clone https://github.com/nate-sys/sycamore-journal
```
or checkout the [Live Demo](https://nate-sys.github.io/jwapp)

## Dendencies
* Rust (Cargo)
* [Sycamore](sycamore-rs.netlify.app)
